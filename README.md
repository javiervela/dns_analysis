# DNS Analysis

Analysis of DNS records using a Python client that wraps the dig CLI tool.

See [report.pdf](doc/report.pdf) for details.

## Project Structure

```txt
 dns_analysis

 ├── scripts                # Contains all experiment scripts
 ├── src
 │   └── dnsclient
 │       └── __init__.py    # DNS client implementation
 └── uv.lock
 ```
